# Access Mod Template

Makes games playable for blind players through structured mod development.

## Features

- Claude Code integration for AI-assisted development
- Documented patterns for UI navigation, input handling, and audio feedback
- Templates and utilities for common accessibility features
- Step-by-step setup guide
- Localization support

## What This Template Provides

- **Code Patterns:** Proven architectural patterns for modular, maintainable accessibility mods
- **Utilities:** Ready-to-use classes for announcements, key handling, and UI extraction
- **Documentation:** Comprehensive guides for common accessibility challenges
- **Templates:** Starter code for menus, navigation, and game state tracking
- **AI Integration:** CLAUDE.md instructions for Claude Code to assist development

## Project Structure

- `docs/` - Guides, checklists, and API documentation
- `templates/` - Code templates for common features
- `scripts/` - PowerShell helper scripts
- `decompiled/` - Game source code (you add this during setup)
- `CLAUDE.md` - Instructions for Claude Code integration

## Getting Started

1. Clone this template
2. Run the setup interview with Claude Code
3. Follow the patterns in `docs/ACCESSIBILITY_MODDING_GUIDE.md`
4. Build your first accessible game feature

## Requirements

- Windows with Git Bash
- .NET SDK
- Screen reader (NVDA, JAWS, or similar)
- Game to mod (Unity/C# based games work best)

## Documentation

- `docs/setup-guide.md` - Initial project setup
- `docs/ACCESSIBILITY_MODDING_GUIDE.md` - Core development patterns
- `docs/game-api.md` - Safe key bindings and common methods
- `docs/localization-guide.md` - Multi-language support
- `docs/menu-accessibility-checklist.md` - Menu implementation guide
